# 簽到機器人
---
優點:
---
完善的指令</br>中文的提示...</br></br></br>

如何設定?</br>
1.於config.json填入資料</br>
> owner_id填你的id</br>
> prefix填你的前綴</br>
> token就填token</br>
> max填入簽到最大金額</br>
> min填入簽到最小金額</br>
> new填入創建帳號初始金額
</br>
2.執行 `bot.py`

</br></br></br>

---
指令大全
---

`help`  __幫助列表__
</br>

`new` __創建帳號__
</br>

`sign` __簽到__
</br>

`me` __看自己有多少錢__
</br>

`money <id>` __看別人有多少錢__
</br>

`get <序號>` __取得序號__
</br>

`to <id> <錢>` __轉帳給別人__
</br></br>

管理指令:</br>
`set <id>` __將ID設為指定金錢__</br>
`code <錢>` __設定兌換代碼__</br>
`dlt <代碼>` __刪除兌換代碼__
